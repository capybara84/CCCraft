# SPEC - Phase 3: ブロック操作

## 概要
レイキャストでブロックを選択し、破壊・設置ができるようにする。ドロップアイテムとインベントリ（ホットバー＋バックパック）も実装する。

## 機能要件

### 1. レイキャストによるブロック選択
- 画面中央のカーソル（十字）からカメラ方向にレイキャスト
- ヒットしたブロックをワイヤーフレームでハイライト表示
- 操作範囲: プレイヤーから半径5ブロック以内

### 2. ブロック破壊
- 左クリック長押しでブロック破壊
- ブロックの硬度に応じた破壊時間（GDD準拠）:
  - 土/草/砂: 0.5秒、葉: 0.3秒、木の幹/木材: 1.0秒、石: 1.5秒、石レンガ: 2.0秒
- 破壊中の進捗バー表示（対象ブロック上に表示）
- 破壊完了でブロックが空気に置き換わり、ドロップアイテムが出現
- ドロップアイテムはGDD準拠（草→土、木の幹→木材など）

### 3. ブロック設置
- 右クリックでブロック設置
- 設置先はレイキャストのhit normalで決定（対象ブロックの隣接面）
- ホットバーで選択中のブロックを設置
- プレイヤーと重なる位置には設置不可

### 4. インベントリ
- ホットバー: 8スロット（画面下部に常時表示）
- バックパック: 24スロット（Eキーで開閉）
- 各スロット: アイテム種 + スタック数（最大64）
- 数字キー1〜8でホットバーのスロット選択
- マウスホイールでホットバーのスロット切替

### 5. HUD
- 画面中央: 十字カーソル
- 画面下部: ホットバー（選択中のスロットをハイライト）

## 非機能要件
- TypeScript strict mode
- マジックナンバーは全て constants.ts に定義
- コメントは日本語

```

│   └── ItemStack.ts            ← アイテムスタック定義（新規）
├── ui/

│   └── InventoryUI.ts          ← バックパックUI（新規）

```

## 完了基準
- 画面中央のカーソルでブロックをハイライト選択できる
- 左クリック長押しでブロックが破壊され、ドロップアイテムがインベントリに入る
- 右クリックでホットバー選択中のブロックを設置できる
- ホットバーが画面下部に表示され、数字キー/ホイールで切替できる
- Eキーでバックパックが開閉できる
