# SPEC - Phase 2: 地形生成

## 概要
Simplex Noiseによる地形生成、チャンクシステム、Greedy Meshing、3種バイオームを実装し、複数の浮島が空中に浮かぶワールドを作る。

## 機能要件

### 1. チャンクシステム
- チャンクサイズ: 16×16×16ブロック
- チャンク座標 = `(chunkX * 16, chunkY * 16, chunkZ * 16)`
- 各チャンクは3D配列（Uint8Array）でブロックIDを保持
- プレイヤー周囲のチャンクのみロード/メッシュ生成
- 描画距離: 8チャンク（128ブロック）※パフォーマンスに応じて調整

### 2. Greedy Meshing
- 同種ブロックの隣接面を結合してポリゴン数を削減
- 空気ブロックに隣接する面のみ描画（内部面は除去）
- 各チャンクごとにメッシュを生成

### 3. 地形生成（Simplex Noise）
- simplex-noise npmパッケージを使用
- 2Dノイズ4オクターブで島の形状（密度マップ）を生成
- 3Dノイズで底面を不規則に削り、浮遊感を出す
- 浮島はY=20〜Y=100の範囲に生成

### 4. 複数の浮島
- 3〜5島を生成
- 島のサイズバリエーション: 小（半径8〜15）、中（半径20〜40）、大（半径50〜80）
- 島と島の間隔: 最小10ブロック、最大50ブロック
- プレイヤーは最初の島（草原、中サイズ）にスポーン

### 5. バイオーム（3種）
- 別のノイズレイヤーでバイオームを割り当て
- 草原: 草ブロック地表、なだらかな丘陵、オークの木
- 森: 暗い草ブロック、密集した大木、巨大樹
- 砂漠: 砂ブロック、平坦、サボテン・枯れ木
- バイオームごとにテクスチャカラーと地形特徴が異なる

### 6. 木・植生の生成
- 草原: オークの木（幹3〜5ブロック + 葉の球体）
- 森: 巨大樹（幹5〜8ブロック + 大きな葉）
- 砂漠: サボテン（3〜4ブロックの柱）、枯れ木

## 非機能要件
- TypeScript strict mode
- マジックナンバーは全て constants.ts に定義
- コメントは日本語
- 60fps維持（デスクトップChrome）

## 新規依存パッケージ
- `simplex-noise`: 地形生成用ノイズ

## 完了基準
- 複数の浮島が空中に浮かんでいる
- 各島に異なるバイオーム（草原・森・砂漠）が適用されている
- 木やサボテンなどの植生が生成されている
- Greedy Meshingでパフォーマンスが確保されている
- プレイヤーが各島の上を歩ける（チャンクベースの衝突）
- 島の底面が不規則に削られて浮遊感がある
